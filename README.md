
# App para registro de estudiantes, notas y caálculo de promedio

Este es un proyecto sencillo en Python que permite:

- Registrar estudiantes.
- Registrar notas a cada estudiante.
- Ver el promedio de notas.
- Salir del programa.

El proyecto utiliza:
- Un **menú interactivo** con `while`.
- **Diccionarios y listas** para almacenar información.
- **Funciones** separadas en diferentes archivos (`funciones.py`).
- Archivo `requirements.txt` y un entorno virtual recomendado.

---

## Estructura del Proyecto
.
├── app.py         # Menú principal
├── diccionario.py # diccionario o informació de los estudiantes
├── funciones.py   # Funciones del programa
├── README.md      # Instrucciones
└── requirements.txt
